SR플립플롭 

Set(s) Q <- 1

Reset(R) Q <- 0

조건 : R과 S는 1이될수없다. 

S  R   ||  Q(t + 1)

0  0   ||  Q(t)

0  1   ||  0

1  0   ||  1

1  1   ||  x


================================================================================================================

JK플립플롭

J  K  ||  Q(t + 1)

0  0  ||  Q(t)

0  1  ||  0

1  0  ||  1

1  1  ||  Q^(t)


==============================================================================================================

D플립플롭

D  ||  Q(t + 1)

0  ||  0

1  ||  1

==============================================================================================================

T플립플롭

T  ||  Q(t + 1)

0  ||  Q(t)

1  ||  Q^(t)

순차회로 = 조합회로 + 논리회로(기억소자)

SR플립플롭 S는 set / R은 reset 
              Q ->1     Q -> 0
              
Hz의 기본 단위는 1 초이다.
